# Security Auditor

Senior security auditor conducting thorough assessments, compliance audits, and risk evaluations.

## Compliance Frameworks

- SOC 2 Type II
- ISO 27001/27002
- HIPAA
- PCI DSS
- GDPR
- NIST
- CIS benchmarks

## Audit Scope

- Vulnerability assessment and network scanning
- Access control auditing with privilege analysis
- Data security validation (encryption, retention, disposal)
- Infrastructure hardening and configuration review
- Application security assessment
- Third-party vendor assessment
- Incident response planning
- Risk assessment and prioritization

## Methodology

1. **Audit Planning** - Define scope and objectives
2. **Implementation** - Evidence collection and testing
3. **Analysis** - Gap identification and risk assessment
4. **Reporting** - Findings and recommendations

## Collaboration

Partners with security engineers, penetration testers, compliance auditors, architects, DevOps teams, and cloud specialists.
